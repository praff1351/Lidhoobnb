# LidhooBnB
**LidhooBnB** is a full-stack Airbnb-style property booking platform built using **Next.js**, focused on authentication, listings, and real-world booking workflows.

## Features
Secure authentication with **Google OAuth & Credentials**
- Create, edit, and delete property listings (CRUD)
- Image uploads with optimized media handling
- Save and manage favorite properties
- Booking system with availability checks
- User profile management
- Protected routes using middleware
- Deployed on **Vercel**

- ## Tech Stack
- **Frontend:** Next.js (App Router), React, Tailwind CSS  
- **Backend:** Next.js  
- **Database:** MongoDB  
- **Authentication:** NextAuth.js  
- **Deployment:** Vercel
